# Documentación - Sistema POS La Esquinita

Esta carpeta contiene toda la documentación del proyecto.

## Archivos de Documentación

- `REQUISITOS_EXAMEN.md` - Requisitos completos del examen
- `CONFIGURACION_EMAILJS.md` - Configuración del sistema de emails
- `GUIA_DESARROLLO.md` - Guía para desarrolladores
- `API_DOCUMENTATION.md` - Documentación de APIs

## Estructura del Proyecto

```
La_Esquinita/
├── docs/              # Documentación
├── BD/               # Base de datos y scripts
├── tests/            # Tests unitarios e integración
├── api/              # APIs backend
├── controllers/      # Controladores MVC
├── models/           # Modelos de datos
├── modulos/          # Vistas del sistema
├── partials/         # Componentes reutilizables
├── assets/           # CSS, JS, imágenes
└── config/           # Configuraciones
```
